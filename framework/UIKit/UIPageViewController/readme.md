# UIPageViewController

**简介**

它表示页面视图控制器，一个页面视图控制器允许用户在不同的页面内容之间导航，每个页面由它自己的视图控制器管理。导航可以由你应用的程序控制，也可以直接由用户使用手势控制。



**注意**

* 要支持基于手势的导航，必须使用一个数据源对象来提供视图控制器。
* 从iOS6以后，该类才可以被继承。
