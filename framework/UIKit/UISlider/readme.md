# UISlider

**简介**

一个`UISlider`对象表示一个滑动条，用户可以移动滑块来改变滑动条的当前值。



**自定义外观**

可自定义的部分有：

* 最小值和最大值的图像，这些图像位于滑动条控件的两端。
* 轨道图像
* 滑块图像

像轨道图像和滑块，你可以为滑动条的不同状态分配不同的图像。



**注意**

* 滑块控件提供了一组默认的轨道图像和滑块图像，如果你没有指定自定义图像，会自动使用这些图像。
