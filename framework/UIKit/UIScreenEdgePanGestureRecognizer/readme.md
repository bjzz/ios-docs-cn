# UIScreenEdgePanGestureRecognizer

**简介**

该类用于查找靠近屏幕边缘的平移手势，系统在一些场景下，使用屏幕边缘手势来启动视图控制器的转换。创建一个屏幕边缘平移手势识别器后，在将手势识别器绑定到你的视图之前，给手势识别器的`edges`属性设置一个适当的值，该属性指定了手势的开始边缘。



**注意**

* 这个手势识别器会忽略第一次触摸以外的任何触摸。
* 从iOS7.0开始可用。
